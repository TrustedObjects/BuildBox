# Introduction

BuildBox is an environment designed to develop, build and deliver projects.

It provides a standard way to define and manage projects resources.
The goal is to simplify environment setup and build process, and to help development on embedded / cross-compiled projects.

All projects components and implied tools are versioned and tracked by BuildBox, to guarantee delivery reproducibility in the future.
This is helpful to be able to quickly deliver a new release for an old project, starting from a known state.

As BuildBox projects are running in containers, they are isolated from developer's host system.
